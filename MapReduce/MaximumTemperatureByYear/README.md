# MapReduce Code to Find the Maximum Temperature by Year

### Input Data

The input data contains Date(dd/mm/yyyy), ID, and Temperature
```
10-01-1990,123112,10
14-02-1991,283901,11
10-04-1990,381920,15
10-01-1991,302918,22
12-02-1990,384901,9
```

### Expected Output

```
1990    15
1991    22
```
